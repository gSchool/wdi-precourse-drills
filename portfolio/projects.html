Project 1 - Trip Planner / Weather Reports

In this application, a user clicks on a location on the map which brings up a seven-day weather forecast for that location. By clicking on several locations along the route of a planned trip, the user can find out the probable weather conditions for the trip.

In a future version, the application will calculate the route and travel time based on user input. For example, if the user intends to drive 70 mph for 10 hours a day, and leave Seattle at 12 noon, the application will calculate approximate arrival time in Miami or other destination.

Project 2 - Interactive Map of Apple Orhard

This map shows locations for hundreds of fruit trees. When the user clicks on a tree, an info window pops up with details about the tree, including species, variety, date planted, row number, and notes. The next version will allow orchard managers to add, edit, and delete trees.

Future versions of the project will allow users to access a calendar and sign up for work opportunities. Orchard managers will be able to easily send email alerts to certain categories of users, such as to inform about a tree-planting work day.

Project 3 - Mountain Puzzle

A drag-and-drop puzzle in which users attempt to reassemble a map of a mysterious mountain region. The application tracks the total number of moves for each game.

A future version will also track the time elapsed to complete the puzzle. And it will compare this users time with the best times by previous users.
